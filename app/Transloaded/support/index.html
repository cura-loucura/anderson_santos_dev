Support

Need help with Transloaded? Check the tips below first — most issues are solved in seconds.

Before you write

⬇️

Language packs not installed? Go to System Settings → General → Language & Region → Translation Languages and download the languages you need.

🖥️

App won't open? Transloaded requires macOS 15 Sequoia or later. Check your macOS version in → About This Mac.

📄

PDF not translating? Scanned PDFs need OCR first. Make sure your file isn't password-protected and try dropping it again.
